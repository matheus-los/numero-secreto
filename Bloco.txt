alguns codigos do git para lembrar
git init
git add (adiciona o arquivo)
git branch (seleciona a ramificacao)
git push (atualiza o arquivo no GitHub)
git remote (mostra qual o repositorio remoto que está sendo atualizado)
git commit (adiciona a mensagem de commit no arquivo)
    --amend (altera a mensagem do ultimo commit)
git pull (puxa os arquivos alterados para o repositorio local em caso de colaboração de outra pessoa)
git log (informa no terminal quais as alterações feitas)
git status (mostra os arquivos que foram alterados e se já foram salvos ou não para encaminhar para o GitHub)
git config --global (configura qual usuario esta fazendo as alterações)
git revert (revert commit realizado de acordo com o ID do mesmo)
git reset --hard (apagar commit)
